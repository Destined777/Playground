# 上传任务
如何上传自己完成的任务？
1. Fork本仓库，然后在自己fork的仓库中进行相应的提交。
2. 提出issue，要求issue的名字为 TourX(X为第几次任务)-个人姓名。如:`Tour1-zhangsan`，在描述中简述本次任务中完成的内容
3. 提出pr,将你的pr**关联**到issue上去
[注]: 出于仓库的可读性，所有提出的pr并不会被merge，pr仅作review用途。
